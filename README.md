# rssreader
чтениe rss лент

##использование
Вставьтев ваш шаблон следующий код

    {PHP|cot_rss_parse("UrlRSS", "Количество элементов", "Шаблон")}

## Кэш
Для работы с кэшем, должен быть включен системный кэш

    {PHP|cot_rss_parse("UrlRSS", "Количество элементов", "Шаблон")}

А так же необходимо создать директорию

    $cfg['cache_dir']."\rssreader"
